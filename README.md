Website can be found here: [https://grcetmpk.github.io/quarto-website-template/](https://grcetmpk.github.io/quarto-website-template/)

To launch your website from a GitHub repository (like this one), go to Settings -> Pages (left hand side)

Select "Deploy from a Branch". Select the website to build from the `main` branch, and choose the `/docs` folder.
